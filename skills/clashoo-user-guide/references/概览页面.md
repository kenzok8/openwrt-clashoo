# 概览页面

`概览` 是 Clashoo 的主操作页，主要用于启动服务、切换内核、切换代理模式和查看运行状态。

## 运行状态

运行状态会显示当前服务是否启动，以及当前内核。

常见显示：

- `运行中`：核心进程已启动
- `已停止`：核心进程未运行
- `健康检查失败`：服务启动了，但 DNS、显式代理或透明代理链路有问题
- `降级运行`：选择了 TUN / Mixed，但设备不支持 TUN，系统自动回退

## 内核切换

可选内核：

- `Mihomo`
- `Smart`
- `Sing-box`

切换内核时，Clashoo 会根据当前配置重新准备运行文件。

如果切到 sing-box，请确认已经选择了 sing-box JSON 配置文件。

## 透明代理

透明代理区域显示当前 TCP、UDP 和网络栈模式。

常见组合：

- Fake-IP：TCP 通常为 `Redirect`，UDP 通常为 `TProxy`
- TUN：TCP / UDP 由 TUN 接管
- Mixed：使用 TUN mixed stack

如果设备不支持 TUN，会看到降级提示。

## 访问检查

访问检查用于快速判断国内外访问是否正常。

通常包括：

- 国内站点检查
- YouTube 或 Google 类外站检查
- 上次检查时间

显示 `Timeout` 不一定代表配置文件错误，也可能是当前节点不可用、DNS 异常或网络环境阻断。

## 流量监控

流量监控显示：

- 上传速度
- 下载速度
- 当前连接数
- 内存使用

刚启动时这些数值可能为 0，访问网页后会变化。

## 底部快捷设置

概览底部可以快速切换：

- 代理模式
- 运行模式
- 配置文件
- 管理面板

这些设置会写入 UCI 配置，并影响下一次运行配置生成。
